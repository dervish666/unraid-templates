# Unraid Docker Templates

Docker templates for Unraid Community Applications.

## Available Templates

### UnraidMonitorBot

Telegram bot for monitoring Docker containers and Unraid servers.

- **Repository:** [dervish/unraidmonitorbot](https://hub.docker.com/r/dervish/unraidmonitorbot)
- **Project:** [GitHub](https://github.com/dervish666/UnraidMonitor)
- **Support:** [Issues](https://github.com/dervish666/UnraidMonitor/issues)

## Installation

These templates are available through Unraid's Community Applications plugin.

1. Install the Community Applications plugin from Unraid's Apps tab
2. Search for "UnraidMonitorBot"
3. Click Install and configure the required variables
